## 自用代码
>一些在不同地方修改网页可能会用到的代码   im a noob so 原谅我整点help      

##### 添加所有变更到暂存区
```bash
git add --all
```
##### 更改提交，添加描述
```bash
git commit -m '描述'
```
##### 推送到GitHub
```bash
git push origin main
```
##### 从GitHub拉取更新
```bash
git pull origin main
```
